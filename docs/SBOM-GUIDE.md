# Nerion SBOM Generation Guide

## TypeScript SBOM (SPDX via REUSE)
The SPDX SBOM is generated by REUSE:
```bash
reuse spdx > nerion.spdx
```
This produces nerion.spdx from the REUSE.toml annotations.
The file is regenerated on every release (not committed to git — add to .gitignore if large).

## Rust SBOM (CycloneDX via cargo-cyclonedx)
```bash
cd rust && cargo cyclonedx --format json --output-cdx
```
Produces rust/bom.cdx.json. See ci-sbom.yml for CI generation.

## SPDX License Identifiers
All source files carry SPDX identifiers per REUSE 3.3. Run: reuse lint
